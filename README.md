## SecureAuth App
### 😃 Description:

A comprehensive authentication application that ensures secure user registration and login. 

### ⚠ Packages Used:

1. Node.js
2. Express
3. Passport
4. EJS

### 🚀 Usage:

```sh
$ npm install
```
```sh
$ npm start
# Or run with Nodemon
$ nmp run dev

#Visit http://localhost:5000
```
##### 🍃 MongoDB:

Open "config/keys.js" and add your MongoDB URI, local or Atlas
